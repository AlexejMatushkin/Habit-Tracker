# Habit Tracker + Pomodoro Timer

Приложение для осознанного цифрового детокса:
- Учёт ежедневных привычек с сохранением между запусками
- Pomodoro-таймер (25/5) для концентрации внимания
- Полностью offline-first, без интернета

## Технологии
- Kotlin, Jetpack Compose, Coroutines, Flow
- Room Database, Clean Architecture (MVVM)
- Navigation Compose, ViewModel, State

## Особенности
- Реактивный UI: изменения мгновенно отражаются в интерфейсе
- Данные хранятся локально — никаких облаков или аккаунтов
- Чистая архитектура: легко расширять (добавить экспорт, уведомления, статистику)
